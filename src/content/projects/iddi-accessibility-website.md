## 背景

遊石設計參與經濟部工業局 IDDI 設計驅動產業創新補助計畫,透過建置公司的無障礙網頁,宣傳公司在計畫中的產出成果,作為遊石設計在無障礙領域的展示、宣傳與溝通平台。經費不足且公司人力短缺,由我一人身兼專案管理與 UI/UX,並與開發端對接,同時要滿足無障礙設計規範。

## 專案內容

耗時四個月,基於遊石設計新官網的視覺風格,從零製作一個 landing page 與兩組功能模塊。

## 專案挑戰

在資源有限、一人多角的情況下,既要確保網站無障礙設計符合 WCAG 指南,又要持續確認開發進度並調整開發細節,還要與同計畫下不同 PM 協調跨時差溝通。

## 個人貢獻

- 針對網站行銷定位進行桌面研究,分析競品及蒐集相關資料。
- 規劃並管理設計師工作項目和排程,定期舉行例會。
- 製作網站所有頁面與互動細節、GUI 與 design guideline。
- 確保網站無障礙設計符合指南,持續確認開發進度並調整開發細節。

## 專案結果與影響

完整體驗了產品從零到一開發一條龍的流程,以及一人身兼多角的專案運作;透過與同計畫下不同 PM 的溝通協調,掌握了跨時差溝通與多重工項間的協調能力。此網站後來也作為北亞利桑那大學 IDDI 計畫下無障礙推廣網站的實作基礎,採 React 與 Tailwind CSS 重建,符合 WCAG 標準。
