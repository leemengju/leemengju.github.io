## 背景

本專案為資展國際全端工程師養成班 PHPD09 的畢業專題,採五人協作模式。多數組員缺乏實際開發經驗,需要在學習新技術的同時完成專題開發,並準備面試筆試審查,時間壓力巨大;專案過程中幾乎每天開會確認功能需求、同步進度、協作除錯與整合,對新手工程師的抗壓性與自主學習能力是很大的挑戰。

## 專案內容

前端使用 React、Vite、Tailwind 和 Shadcn UI,後端採 Laravel(PHP)框架,資料存於 MySQL,並以 GitHub 進行版本控制,最終完成電商平台的客戶端與企業端功能開發。

## 專案挑戰

多數團隊成員第一次接觸真正的多人協作開發,需要在陌生的技術棧上同時處理前後端整合、API 串接與資料庫設計,並在有限時間內對齊彼此的開發進度與程式風格。

## 個人貢獻

- 完成網站規劃及客戶端購物車結帳功能開發,成功串接資料庫與 API,確保功能順暢運行。
- 開發並優化企業端的訂單管理與系統設定功能,實現訂單處理操作介面,提升系統管理效率與使用體驗。
- 統整全系統 UI 設計,並帶領團隊完成專案開發,最終完成畢業簡報與匯報,呈現專案成果與技術細節。

## 專案結果與影響

儘管面臨時間壓力與團隊經驗落差,最終成功交付 MVP,客戶端與企業端系統可獨立運行;透過 API 串接完成訂單資訊、商品賣場同步更新、行銷活動、金流交易、會員管理及系統設定等功能,專案 Demo 獲講師團隊高度評價。
