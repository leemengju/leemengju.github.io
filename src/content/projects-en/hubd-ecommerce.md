## Background

This project was the capstone (PHPD09) of iSpan International's full-stack engineer training program, completed by a 5-person team. Most members lacked real development experience and had to learn new technologies while building the capstone and preparing for interview written-test reviews, under heavy time pressure; the team met almost daily to confirm feature requirements, sync progress, and collaborate on debugging and integration — a major test of novice engineers' resilience under pressure and self-directed learning.

## Scope

Built the front end with React, Vite, Tailwind and Shadcn UI, with Laravel (PHP) as the back-end framework, MySQL for data storage, and GitHub for version control — ultimately delivering both the customer-side and enterprise-side features of the e-commerce platform.

## Challenges

Most team members were experiencing real multi-person collaborative development for the first time, having to handle front/back-end integration, API integration and database design on an unfamiliar tech stack while aligning each other's development progress and coding style within limited time.

## Contributions

- Delivered site planning and the customer-side cart and checkout flow, successfully integrating the database and APIs to ensure smooth operation.
- Built and optimized the enterprise-side order management and system settings features, implementing the order-processing interface to improve admin efficiency and user experience.
- Consolidated the whole-system UI design and led the team through project completion, delivering the final graduation presentation covering the project results and technical details.

## Impact

Despite the time pressure and the team's experience gap, we successfully delivered the MVP, with the customer-side and enterprise-side systems running independently; API integration enabled order information, catalog synchronization, promotions, payment transactions, membership management and system settings, and the project demo was highly praised by the instructor team.
